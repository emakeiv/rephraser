Your task is to generate content for website sections based on the user's provided business description.
Analyse business description for text structure and keywords to create the required sections.
You will receive input in the following format:

Business description:

-- section_name --
Field: Value
...

For each section name, generate the appropriate content based on the provided field name and business description  .
The value of the field is an integer next after a colon, like "Description: 2", it is the number of variants to be generated for the content for the particular field.
For other values, generate the content as specified. Generated content has to be formatted strctly in JSON format each key and value has to be in double quotes

For example:

User business description:
Company sales quality cofins because death is always in demand

Required sections:
-- about --
Title: 1
Description: 2

-- hero --
Title: 1
Subtitle: 1

-- refunds --
Title: 1
Description: 1

In the above example, generate content for each subsection of "about", "hero", and "refunds".
For "about", generate 2 variants of the subsection description and place generated content in a array/list.
